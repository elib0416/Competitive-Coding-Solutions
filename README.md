# Competitive-Coding-Solutions
These are my solutions to competitive coding problems from various contests. I hope this inspires you to try them out!

## File Naming Convention

Files follow the format: `ProblemName_Source.cpp`

The suffix after the underscore indicates the source and/or division:

| Suffix | Meaning |
|--------|---------|
| `_Bronze` | USACO Bronze division |
| `_Silver` | USACO Silver division |
| `_Gold` | USACO Gold division |
| `_OldBronze`, `_OldSilver`, `_OldGold` | Older USACO problems |
| `_CSES` | [CSES Problem Set](https://cses.fi/problemset/) |
| `_CF` | Codeforces |
| `_LIT` | Lexington Informatics Tournament |
| `_BAPC<year>Novice` | Bay Area Programming Contest Novice division |

Some older files use lowercase names without a suffix — these are named after the USACO input/output file (e.g., the problem's `.in`/`.out` filename).
